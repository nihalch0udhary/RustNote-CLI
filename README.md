## Project Title
RustNote CLI

## Project Description
RustNote CLI is a fast command line notes manager that stores notes as JSON in the user home folder. It supports adding quick notes, listing stored notes, searching by text and removing notes by id.

## Project Vision
The vision is to provide a small, fast and dependency light notes tool for programmers who prefer keyboard centric workflows and file based persistence.

## Key Features
Add notes with title and body from the command line. List notes with basic metadata. Search notes by title or body content. Remove a note by id. Notes are stored in a single JSON file under the user home folder.

## Future Scope
Add tagging and notebooks, full text indexing for faster searches and encryption of stored notes. Add import and export tools and a simple TUI for browsing and editing notes.
